1.	Add a navigation menu to your resume site.
			      a.	For this HW add a navigation menu without using a list. An example of this is can be found in the lecture material. 
EX 1a html: <nav class="display">
  <a href="#">About</a>
  <a href="#">Work</a>
  <a href="#">Education</a>
  <a href="#">Contact</a>
</nav>

EX 1a css:

/* General styles */
html {
  box-sizing: border-box;
}
*, *:before, *:after {
  box-sizing: inherit;
}
nav {
  background: purple;
  margin-bottom: 5px;
}
a {
  background: lightblue;
}

/* Navigation using 'display' */
.display {
  text-align: center;
  
  /* To reduce the space between the elements set the font size to zero in the parent element. */
  
  /* font-size: 0; */
  
}
.display a {
  padding: 10px;
  display: inline-block;
  
  /* This extra space affects how much total space the element takes up. Let's say I want to make four equal elements, by setting the links to a width of 25%. Even though 25 x 4 = 100%, the last item gets pushed down to the next line because there's not enough space for it. */  
  
  /* width: 25%; */
  
  
  
  /* Because font size is also inherited, you'll have to reset the font size in the child element itself. */
  
  /* font-size: 16px; */
}
